# flixster

A Rails video-streaming-course application (Udemy-style), built late 2014 as a multi-model Rails practice project.

## Stack

- Ruby on Rails 4
- CarrierWave for image and video uploads
- SASS
- `video-js` for the player

## Domain

Courses, Sections, Lessons, Enrollments, Users. Instructor namespace for course authoring; public namespace for browsing and enrollment.

## Context

Same 2014-2015 learning arc as `drunken-octo-spice` and `scaling-hipster`. Not deployed. Not maintained since 2015.
